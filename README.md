## â“ Troubleshooting

### White Screen / Clerk Failed to Load

**Problem:** Site shows white, console shows Clerk error

**Solution:**
1. Check if Clerk key is set in `.env.local`
2. Verify key is REAL (not base64 encoded)
3. Key should be 60+ chars, start with `pk_test_`
4. No spaces or line breaks in key
5. Restart dev server: `npm run dev`

See [CLERK_QUICK_FIX.md](./CLERK_QUICK_FIX.md) for detailed troubleshooting

### Backend API Not Responding

**Problem:** Frontend calls `/api/posts` but gets 404 or connection refused

**Solution:**
1. Check backend is running: `python manage.py runserver`
2. Check port 8000 is open
3. Check `VITE_API_URL` in `.env.local` is correct
4. Try calling API directly: `curl http://localhost:8000/api/posts`

### Build Fails on Vercel

**Problem:** GitHub Actions workflow fails during build

**Solution:**
1. Check GitHub Secrets are set correctly
2. Check Clerk key is not truncated
3. Check build logs: GitHub â†’ Actions â†’ Failed workflow
4. Look for: missing packages, TypeScript errors, or invalid env vars
